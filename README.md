# MoovieTime

## Getting Started

Make sure to install the dependencies:

```bash
# npm
npm install
```

## Development Server

Start the development server on <http://localhost:3000>

```bash
npm run dev
```

## Production

Build the application for production:

```bash
npm run build
```

Locally preview production build:

```bash
npm run preview
```

Notes:

- Node Version (v18.13.0)
- npm version  (8.9.3)
